What is Scope in JS and How many types?

Scope mean in js an area or space where you can access a variable and Js have three types of Scope that are

  • Global Scope
  • Functional Scope
  • Block Scope

    • Global Scope:

    Global scope mean in js when you declare a variable out of functons and blocks than you can access the variable all of the file , from any funciton and block.

    Functional Scope:

    Its mean when you declare a variable in funciton, then you can access the variable from any place of that funciton, it is called funcitonal scope.

    Block Scope:

    Block scope mean in js declare a variable in block like {} if, for then you cannot access the variable out of these block, it is called block scope.
